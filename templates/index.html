Welcome to the Flask Quizzer!

This is a Flask web application! It responds to HTTP requests with JSON objects.

  • app.py: contains the Flask webserver for handling HTTP requests.
  • quizzer.py: provdes the functions for getting questions and checking answers.
  • questions.py: contains only the data, i.e., questions and answers for the quiz.

You can make requests to the following endpoints:

  • /api/question/[int]: /api/question/{{ question.question_id }}
  • /api/question/[int]/answer/[int]:
    • Correct answer: /api/question/{{ question.question_id }}/answer/0
    • Inorrect answer: /api/question/{{ question.question_id }}/answer/1
  • /api/random-question: /api/random-question
